## 准备工作

### 材料

学校会发的材料：面包板、两根 USB 线、FPGA 开发板、芯片若干、导线 2m（今年）。搭接是最痛苦的，导线也存在不够长的情况，有以下方案：

1. 自购杜邦线（公对公）：多几个长度，非常自由，本人选择
2. 自购成品跳线：次选
3. 原有导线 + 自购剥线钳：你会痛苦的，而且大概率导线不够长
