## 准备工作

### 材料

学校会发的材料：面包板、两根 USB 线、FPGA 开发板、芯片若干、导线 2m（今年）。搭接是最痛苦的，开课前一周务必记得购买材料，有以下方案：

- [x] 自购杜邦线（公对公）：多几个长度，非常自由，本人选择

- [ ] 自购成品跳线：优选

- [ ] 原有导线 + 自购剥线钳：你会痛苦的，而且大概率导线不够长

#### draw.io 使用

打开文件：菜单 — 文件 — 从...打开 — 设备 — 下载目录

添加电路包：左侧边栏 — +更多图形 — 电路

#### Fritzing 使用

参考慕课或[教学视频](https://pan.seu.edu.cn/link/AA3A3869F1758240559EDC5B315C9437E7)。

#### Quartus 使用

询问 AI 大人，结合 教学计划.pdf 附录部分。

## 注意事项

1. 部分 USB 线不能成为数据线，导致硬件驱动安装找不到
2. 建议提前搭好通电测试
3. FPGA 实验较为复杂，所有 Quartus 操作都可以询问 AI
4. 不要忘了慕课
